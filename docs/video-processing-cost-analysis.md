**업로드 제한 정책:**

- **최대 영상 길이**: 20분
- **월 업로드 제한**: 크리에이터당 50개

## 💰 핵심 결론 (바쁜 분들을 위해)

**구독자 10명, 월 1개 영상 기준 (1080p DSLR 촬영):**
- **플랫폼 AWS 운영비용**: 3,510원 (CDN 적용시 2,703원)
- **크리에이터 손익분기점**: 월 5개 영상 (강화된 최소 구독료 기준)
- **플랫폼 적자 완전 방지**: 구독료 6,000원 이상 의무 + 구독자 5명 이상
